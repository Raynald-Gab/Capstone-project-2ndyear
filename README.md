## Enrollment System

This is my OOP Enrollment-system

This is my enrollment-system is like other enrollment system that has
the add student, enrollment, courses, Tuition fee and more
___
### Features
- add, update, remove, and view students
- add, update, remove, and view courses
- add instructors and assign them to sections
- enroll students in sections (may capacity check!)
- view department hierarchy (department > sections > students)
- compute tuition fee (1500 per unit) and process payments

---
### Step By Step of the enrollement 

1. Manage Students 
   - [1] Add student ->  Mag ge-generate ng Student ID after you put your name and your Email
   - [2] Update - Update lng yung name 
   - [3] Remove -> remove 
   - [4] View all -> Back


2. Manage Course 
   - -> makikita mo here yung Courses, tuition and profs 


3. Manage Instructors
   -You can see here the name of your Professor and ano yung handle nilang subject 


4. Enroll Student in Section 
   - Student ID  -> Enter the Student ID That is provided to you 
   - Sections -> Choose which section 
   


5. View Department Hierarchy
   - ipapakita here kung ilan na yung mga enrolled sa mga section and kung ilan nalang ang slot na available


6. Tuition and Payment
   - [1] Calculate Fee 
     - -> Student ID: -> Tuition Price
   - [2] Make Payment 
     - Student ID: -> Payment -> balance Remaining 
   - [3] Check Balance
     - Student ID -> Shown the remaining balance
